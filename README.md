So, I have made a programming language that I call xenon. It is a simple python-like and lua-like language that you can compile using the xenc binary.

Also, you can find the package manager, foundation, for xenon at: https://github.com/Mebecool1/foundation-packages
There, you can find all the packages you need or you can also install the foundation.exe in releases, which is the main package manager and installs or updates packages depending on if you have the package or not.
Rest of the information for foundation will be there at the given link.

Also, you can find the build system, lbmake, for xenon at: https://github.com/Mebecool1/lbmake

The precompiled binary of xenc (the xenon compiler) can be found in the main tree only.

You can use ./install.sh, ./remove.sh, and ./build.sh for convenience.
